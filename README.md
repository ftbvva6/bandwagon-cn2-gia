## 先搞清楚：电信直连是什么，为什么值得多花这个钱

所谓"电信直连"，在搬瓦工这里指的是走中国电信 CN2 GIA 线路——正式名称是 CTGNet CN2 GIA（AS4809），这是中国电信跨国线路里质量最高的那一档。

和普通 163 骨干线路比，CN2 GIA 走的是专用通道，不和普通互联网流量混一起排队，晚高峰抗拥堵能力强得多。搬瓦工官方把它定位成"企业级传输（Enterprise grade transport）"，这个说法基本准确——你的数据从机房出发到中国境内，全程都是这条高优先级专线，不会在中间某个节点突然撞上普通拥堵。

顺便说一下 CN2 GT 和 CN2 GIA 的区别：GT 是 Global Transit，部分节点会走普通线路，高峰期容易堵；GIA 是 Global Internet Access，全程 GIA 专线，稳定性明显更好。搬瓦工目前走全程 CN2 GIA 的套餐，主要是 CN2 GIA-E 系列和香港 / 日本东京 CN2 GIA 系列。

代价是价格贵很多。香港是 $89.99/月起，比 CN2 GIA-E 季度价格贵了两倍多。
